    你好，2020（下）
    发表于 2020-07-14| 更新于 2020-07-15|生活
    字数总计:1.1k|阅读时长: 3 分钟
    |阅读量:|评论数:

    Hello, this is a new era.

    你好，七月

    今天是7月15日，不知不觉中2020年的7月已经过半。

    时间总是悄无声息地在你我的掌中溜走，等到我们发觉，回首一看，走过的路已经快要被尘土覆盖。

    所以，为了让自己的时间过得不那么得容易被遗忘，同时也是改一改自己健忘的毛病，我决定将2020年的规划写在纸上，让自己的未来有所期待，让自己的过去有所追寻。

    关于规划

    规划分为长期和短期，对于2020年的下半年来说，应该算是短期规划，但是又不想让自己的眼光显得太狭窄，所以还是长远都规划一下，规划得要长远。

    长期规划

    说起长期规划，作为一个来到这个世上仅有25年的人来说，可能不一定看得这么清晰。

    但是对于目前的我来说，我希望在我30岁之前，也就是距离现在5年以后，能够实现这么几个小目标：

    1. 找到自己热爱的事业并且全身心的投入进去：之所以事业在我的list里面排在第一位，并不是说其他事情不重要，而是说我认为作为三十而立的年纪，只有将自己的事业确定了，才能在爱情、家庭等方面有足够的支撑，否则只是盯着个人感情，就丧失了对事业和自我价值的追求；
    2. 和我现在所爱的人结婚并且有一个宝宝：按照目前的规划来说，2020年应该是我们结婚的年份，然后我们的事业再奋斗两年，等到2022年的时候我的五年社保交齐，她也正好到30岁的年纪，那个时候我们应该有能力去要一个小宝宝。
    3. 在北京这座城市扎下脚跟，这个扎下脚跟一方面指自己的事业或者工作在北京是能够容身的，另外一点是希望我们在30岁这个年纪能够在北京有一个自己的房子以及我们自己的一辆车，同时并不会因为房子和车而压缩生活的质量。如果从这样的目标来说，我们到时候两个人加一块的月薪应该至少需要7万=3万的房贷+1.5万的生活开销+2.5万的存款。
    4. 财富方面其实很难实现财富自由，但是还是希望自己到那个年纪，能够不再太为金钱发愁。其实从某种角度来说，现在的社会要跨越阶层 ，实现财富自由，既容易也不容易。容易点在于机会确实还有很多，但是能够挖掘且坚持的人不多，不容易点在于对于社会绝大多数人来说是不太可能实现的目标。所以，希望通过五年的努力，这个努力并不是工作上的努力，而是自己挖掘一些其他部分的内容，能够实现这个目标。

    短期规划

    短期规划需要先定义一个时间段，目前来说就先定义到7月底，还有半个月的时间。

    1. 减肥：目前的体重大约在167斤左右，目标经过15天，在7月底能够到达160公斤的水平；
    2. 习惯养成：在7月底前将阅读、写作、锻炼的习惯养成，坚持每天进行相应的部分；
    3. 工作梳理：对于目前的工作可以梳理的比较清楚，各方面的流程和安排比较清晰，同时人员也可以到位，确保工作内容不出问题。
    4. 考试和学业：7月底前想清楚自己自己规划的第一版，以及自己对于未来职业规划，现在可以提前先进行的内容，不管是对于行业的研究，还是对于一些证书的准备，都是可以在目前这个阶段提前开始的。同时关于学业方面的安排，我也希望能够在7月份就定下来，提前规划，提前准备。

    最后

    规划都是次要的，最重要的还是要坚持。将规划落实到自己每一天的生活中去，将战略拆解到每一步可执行的行动中来。
